= {TSFName} Workshop

Welcome to the {TSFName} ({TSFShortName}) workshop!

== What you'll learn

In this workshop, you will learn how to deploy and use {TSFName}, a secure software supply chain solution for {OCPName}. By the end of this workshop, you will be able to:

* Prepare your {OCPName} cluster for {TSFShortName} installation
* Install and configure {TSFShortName} with GitHub or GitLab integration
* Verify your deployment and access the Konflux UI
* Build, sign, and verify your first container image with SLSA Level 3 provenance
* Understand how {TSFShortName} integrates {RHTASName} for cryptographic signing
* Generate and analyze Software Bills of Materials (SBOM) with {RHTPAName}

== Who this is for

This workshop is designed for platform engineers, DevSecOps practitioners, and developers who want to implement secure software supply chain practices on {OCPName}. You should have:

* Basic knowledge of {OCPName} and Kubernetes concepts
* Familiarity with CI/CD pipelines (Tekton)
* Understanding of container images and registries
* Experience with GitHub/GitLab workflows

== What is {TSFName}?

{TSFName} is a deployable instance of Konflux that brings a complete secure software supply chain to your {OCPName} cluster. It integrates multiple Red Hat technologies to enable:

* **Secure build pipelines** -- Tekton-based CI/CD that generates signed container images with SLSA Level 3 provenance
* **Artifact signing** -- Automatic cryptographic signing using {RHTASName}
* **Software bill of materials** -- SBOM generation with vulnerability scanning
* **Identity management** -- Keycloak-based authentication and access control
* **Container registry integration** -- Automated deployment to Quay

== Prerequisites

Before starting this workshop, you should have access to:

* {OCPName} cluster running version {OCPVersion} or later (3+ nodes, not SNO)
* Cluster-admin privileges
* Quay registry account (quay.io or self-hosted)
* GitHub organization OR GitLab instance
* Local workstation with:
** {OCPCli} CLI installed
** Podman installed (Docker not supported)
