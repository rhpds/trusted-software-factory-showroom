== Technical requirements

=== Local workstation tools

Your local machine needs:

* **{OCPCli} CLI:** OpenShift command-line tool installed and configured
* **Podman:** Container runtime (Docker is not supported)
* **Web browser:** Modern browser for accessing Konflux UI
* **Text editor:** For creating and editing configuration files
